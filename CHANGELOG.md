# CoreManager Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-11-18

### Added
- Initial release of CoreManager
- JavaFX desktop application with WebView
- React-based frontend with TailwindCSS styling
- File browsing and management capabilities
- File preview pane for text files
- AI utilities panel with system monitoring
- Plugin system for extensions (Docker, SSH, Monitor)
- Cross-platform support (Linux, Windows, macOS)
- .deb packaging support

### Features
- Modern UI with responsive design
- Real-time system stats (CPU, memory)
- Extensible plugin architecture
- Secure file operations
- Custom theming with SCSS

### Technical
- Java 17 + JavaFX 17 for desktop framework
- React 18 for UI components
- Node.js for build system
- Maven for Java dependency management
- Webpack for frontend bundling
