# CoreManager User Guide

Welcome to CoreManager! This guide will help you get started with using the application.

## Getting Started

### Installation
1. Download the latest release from the [GitHub Releases](https://github.com/paulmmoore3416/corefilemanager/releases) page.
2. Extract the archive to your desired location.
3. Run the executable file.

### System Requirements
- Linux, Windows, or macOS
- At least 4GB RAM recommended
- 500MB free disk space

## Interface Overview

### Main Components
- **Sidebar**: Navigate drives, favorites, and extensions
- **File List**: Browse and manage files in the current directory
- **Preview Pane**: View file contents without opening external applications
- **Toolbar**: Access file operations and settings
- **AI Panel**: Monitor system stats and use AI features

### Navigation
- Use the sidebar to switch between different locations
- Double-click folders in the file list to navigate
- Use the breadcrumb navigation for quick jumps

## File Operations

### Basic Operations
- **Copy**: Select files and use Ctrl+C or the toolbar button
- **Move**: Drag and drop files or use Ctrl+X then Ctrl+V
- **Delete**: Select files and press Delete or use the toolbar
- **Rename**: Right-click a file and select "Rename"

### Advanced Operations
- **Batch Operations**: Select multiple files for bulk actions
- **Search**: Use the search bar for natural language file search
- **Preview**: Click on files to preview in the pane

## AI Features

### System Monitoring
- View real-time CPU, memory, and disk usage
- Monitor network activity
- Get system health insights

### File Analysis
- AI-powered file categorization
- Smart suggestions for file organization
- Content analysis for supported file types

## Extensions

### Built-in Extensions
- **Docker**: Manage containers
- **SSH**: Connect to remote servers
- **Monitor**: Advanced system monitoring

### Custom Extensions
CoreManager supports custom plugins. See the [developer documentation](DEVELOPER_GUIDE.md) for creating extensions.

## Customization

### Themes
- Toggle between light and dark themes
- Use the GitHub Gray theme for a professional look

### Settings
- Access settings through the toolbar
- Configure default views and behaviors

## Troubleshooting

### Common Issues
- **Application won't start**: Ensure you have the required system dependencies
- **Files not showing**: Check file permissions and refresh the view
- **Performance issues**: Close unnecessary applications and restart CoreManager

### Getting Help
- Check the [FAQ](FAQ.md) for common questions
- Report issues on [GitHub Issues](https://github.com/paulmmoore3416/corefilemanager/issues)
- Contact support at support@coremanager.com
