# Frequently Asked Questions

## General

### What is CoreManager?
CoreManager is a modern, AI-enhanced file manager built with Electron and React. It provides an intuitive interface for file management with advanced features like AI-powered search and system monitoring.

### Is CoreManager free?
Yes, CoreManager is open source and free to use under the MIT License.

### What platforms does CoreManager support?
CoreManager runs on Linux, Windows, and macOS.

## Installation

### How do I install CoreManager?
Download the latest release from our [GitHub Releases](https://github.com/paulmmoore3416/corefilemanager/releases) page and extract the archive. No installation is required - just run the executable.

### What are the system requirements?
- Operating System: Linux, Windows, or macOS
- RAM: 4GB minimum, 8GB recommended
- Disk Space: 500MB free space
- Node.js 18+ (for development only)

## Usage

### How do I navigate between folders?
Use the sidebar to select drives or favorites, or double-click folders in the file list.

### Can I preview files?
Yes, click on supported files in the file list to preview them in the preview pane.

### How do I use the AI features?
The AI panel provides system monitoring and file analysis. Access it through the sidebar.

## Troubleshooting

### The application won't start
- Ensure you have the correct system requirements
- Check that all dependencies are installed (for development)
- Try running as administrator/sudo if on Linux/macOS

### Files are not showing up
- Refresh the view with F5
- Check file permissions
- Ensure the drive/folder is accessible

### Performance is slow
- Close other applications
- Restart CoreManager
- Check system resources in the AI panel

### Extensions are not working
- Ensure extensions are in the correct directory
- Check the console for error messages
- Restart the application

## Development

### How do I contribute?
See our [Contributing Guide](../CONTRIBUTING.md) for detailed instructions.

### How do I build from source?
1. Clone the repository
2. Run `npm install`
3. Run `npm run build`
4. Run `npm start`
